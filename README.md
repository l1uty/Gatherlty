一个集合多款工具的自动化信息收集轮子
# 用法
首先打开quake.py文件并在第17行配好自己的360Quake的key，然后安装好依赖并赋予工具执行权限即可 👐

**Usage**
```
# chmod +x wkGdScan.sh
# cd OneForAll
# pip3 install -r requirements.txt
# cd ..
# ./liuty_gather.sh
Usage: ./wkGdScan.sh -u <URL>
# ./liuty_gather.sh -u 主域名
```
注意: 每次收集结束的文件都会覆盖上一次，所以记得注意及时查看result目录下的结果文件🤯
