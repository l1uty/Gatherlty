一个集合多款工具的自动化信息收集轮子
# 用法
首先打开quake.py文件，在第17行配好自己的360Quake的key
然后进入到oneforall目录下，安装依赖：pip3 install -r requirements.txt
再赋予执行权限：chmod +x liuty_gather.sh
最后执行：
./liuty_gather.sh -u 主域名

**Usage**
```
# chmod +x wkGdScan.sh
# cd OneForAll
# pip3 install -r requirements.txt
# cd ..
# ./liuty_gather.sh
Usage: ./wkGdScan.sh -u <URL>
# ./liuty_gather.sh -u 主域名
```
